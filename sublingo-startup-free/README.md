# SubLingo Studio

A startup-style subtitle translator website built with Flask.

## Features
- Drag and drop subtitle upload
- Supports `.srt` and `.vtt`
- Free translation engine using `deep-translator`
- Searchable subtitle block preview
- Optional local video preview
- Download translated file

## Run locally

```bash
pip install -r requirements.txt
python app.py
```

Then open:
http://127.0.0.1:5000
